
# GymApp &emsp;&emsp; <img align="center" src="app/src/main/res/drawable-v24/logo1.png"  height="42">


GymApp is an android application that helps users  plan their workouts and track their progress 

# Introduction
The GymApp application allows the user to conveniently plan and conduct training.
The application was created to free the user from remembering training sessions and writing down the progress in a notebook.
The Gym App allows you to: plan your training, track your progress during training, view the statistics of previously completed training sessions.
